Install XAMPP
	May need to run as administrator depending on system

Install needed services & Start
	Apache
	MySQL

Click on “Admin” button to verify Apache & MySQL servers are running

Click on “Explorer”
	Navigate to xampp > htdocs
		This is where Apache gets the files for hosting
		Make a folder and put html and php in it
			“localhost/[foldername]/    is the url

Navigate to xampp > mysql > data > [database name]
	This is where any data needed for importing (.csv) is stored

After, use the website as any normal website.
